# نمونه کار رضا اسماعیلی - توسعه‌دهنده فرانت‌اند

## ساختار پروژه

این نمونه کار به صورت جداگانه برای هر بخش ایجاد شده است، با رعایت قوانین clean code:

### پوشه‌های اصلی:

- **`home/`** - بخش خانه (صفحه اصلی)
  - `home.html` - فایل HTML بخش خانه
  - `home.css` - فایل استایل بخش خانه
  - `home.js` - فایل جاوااسکریпт بخش خانه

- **`about/`** - بخش درباره من
  - `about.html` - فایل HTML بخش درباره من
  - `about.css` - فایل استایل بخش درباره من
  - `about.js` - فایل جاوااسکریпт بخش درباره من

- **`skills/`** - بخش مهارت‌ها
  - `skills.html` - فایل HTML بخش مهارت‌ها
  - `skills.css` - فایل استایل بخش مهارت‌ها
  - `skills.js` - فایل جاوااسکریпт بخش مهارت‌ها

- **`projects/`** - بخش پروژه‌ها
  - `projects.html` - فایل HTML بخش پروژه‌ها
  - `projects.css` - فایل استایل بخش پروژه‌ها
  - `projects.js` - فایل جاوااسکریпт بخش پروژه‌ها

- **`contact/`** - بخش تماس
  - `contact.html` - فایل HTML بخش تماس
  - `contact.css` - فایل استایل بخش تماس
  - `contact.js` - فایل جاوااسکریпт بخش تماس

### پوشه‌های دیگر:

- **`assets/`** - فایل‌های عمومی
  - `css/` - استایل‌های عمومی
  - `js/` - اسکریپت‌های عمومی

- **`pages/`** - فایل‌های قدیمی (نگه داشته شده برای مرجع)

## ویژگی‌های پروژه

- طراحی واکنش‌گرا
- پشتیبانی از تم تاریک/روشن
- انیمیشن‌های جذاب
- کد نویسی تمیز و ساختارمند
- جداسازی کامل هر بخش

## نحوه استفاده

1. فایل `index.html` را باز کنید
2. از منوی اصلی برای دسترسی به بخش‌های مختلف استفاده کنید

## فناوری‌های استفاده شده

- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- فونت‌های Google
- Font Awesome
