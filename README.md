# نمونه‌کار تفکیک‌شده

این مخزن نسخه ساختارمند نمونه‌کار شخصی را شامل می‌شود که در آن هر بخش در صفحه HTML و فایل جاوااسکریپت اختصاصی خود قرار گرفته است. برای اجرای سریع پروژه کافی است فایل `name.html` را در مرورگر باز کنید تا نمای کلی پروژه به همراه تمام صفحات در دسترس باشد.

## ساختار پوشه‌ها

```
.
├── dist/                # خروجی CSS تولید شده با Tailwind
├── pages/               # صفحات تفکیک‌شده (خانه، درباره، مهارت‌ها، پروژه‌ها، ارتباط)
├── scripts/             # اسکریپت‌های مشترک و مخصوص هر صفحه
├── src/styles/          # فایل ورودی Tailwind و استایل‌های اختصاصی
├── name.html            # صفحه لانچر برای مشاهده همه صفحات در یک قاب
└── index.html           # انتقال خودکار به name.html
```

## راه‌اندازی Tailwind CSS CLI

1. نصب وابستگی‌ها (در صورت دسترسی به اینترنت):

   ```bash
   npm install
   ```

2. ساخت خروجی CSS به‌صورت یک‌باره:

   ```bash
   npm run build:css
   ```

3. اجرای حالت تماشای تغییرات:

   ```bash
   npm run watch:css
   ```

> **نکته:** در محیط فعلی امکان نصب پکیج‌ها وجود نداشت، اما اسکریپت‌ها و پیکربندی لازم فراهم شده است. پس از نصب وابستگی‌ها بر روی سیستم خود، دستورات بالا فایل `dist/styles.css` را با کلاس‌های Tailwind بازتولید می‌کنند.

## نحوه استفاده

- فایل `name.html` را باز کنید تا ناوبری متمرکز به تمام صفحات در یک قاب IFrame در اختیار داشته باشید.
- هر صفحه در پوشه `pages/` نیز به صورت مستقل قابل اجرا است و اسکریپت مخصوص خود را از پوشه `scripts/` فراخوانی می‌کند.
- برای توسعه استایل‌ها کافی است تغییرات را در `src/styles/input.css` اعمال و سپس دستور ساخت Tailwind را اجرا کنید.
