# پورتفولیو حرفه‌ای رضا اسماعیلی

## توضیحات کلی

این یک پورتفولیو حرفه‌ای و مدرن برای نمایش مهارت‌ها و پروژه‌های یک توسعه‌دهنده فرانت‌اند است. پروژه با استفاده از HTML5، CSS3، JavaScript و Tailwind CSS ساخته شده و شامل ویژگی‌های پیشرفته‌ای مانند تم تاریک/روشن، انیمیشن‌های جذاب، افکت‌های تعاملی و طراحی واکنش‌گرا می‌باشد.

## ویژگی‌های پروژه

- **طراحی واکنش‌گرا**: سازگار با تمام اندازه‌های صفحه نمایش از موبایل تا دسکتاپ
- **تم تاریک/روشن**: قابلیت تغییر تم با ذخیره انتخاب کاربر
- **انیمیشن‌های جذاب**: استفاده از Tailwind CSS و CSS Animations برای ایجاد افکت‌های دیداری چشم‌نواز
- **کرسر سفارشی**: طراحی کرسر تعاملی با افکت مغناطیسی روی دکمه‌ها
- **نوار مهارت‌های پویا**: نمایش گرافیکی مهارت‌ها با انیمیشن‌های پویا
- **پروژه‌های قابل اسکرول**: نمایش پروژه‌ها با افکت‌های تعاملی
- **فرم تماس کاربردی**: فرم تماس با اعتبارسنجی سمت کلاینت
- **تایپ‌رایتر پویا**: نمایش نام و توضیحات با افکت تایپ‌رایتر

## ساختار پروژه

```
/workspace/
├── index.html              # فایل اصلی HTML
├── src/                    # پوشه منابع پروژه
│   ├── js/                 # اسکریپت‌های جاوااسکریپت
│   │   └── main.js         # تمام کدهای جاوااسکریپت
│   ├── styles/             # فایل‌های استایل
│   │   └── main.css        # استایل‌های سفارشی
│   └── assets/             # فایل‌های دارایی (تصاویر و ...)
├── README.md               # این فایل
```

## فناوری‌های استفاده شده

- **HTML5**: ساختار اصلی صفحه
- **CSS3 & Tailwind CSS**: استایل‌دهی و طراحی رابط کاربری
- **JavaScript**: تعاملات پویا و عملکردهای سمت کلاینت
- **Font Awesome**: آیکون‌های زیبا و حرفه‌ای
- **Chart.js**: (در صورت نیاز) برای نمایش نمودارهای آماری
- **Vazirmatn Font**: فونت فارسی زیبا و خوانا

## ویژگی‌های پیشرفته

- **انیمیشن‌های حرفه‌ای**: استفاده از keyframes و transitionها برای ایجاد افکت‌های نرم
- **مدیریت تم**: ذخیره و بارگذاری تم انتخابی کاربر با localStorage
- **سیستم اسکرول هوشمند**: انتقال نرم بین بخش‌ها و نشانگر موقعیت فعلی
- **کارایی بالا**: کدنویسی تمیز و بهینه برای عملکرد عالی
- **سازگاری با مرورگرها**: پشتیبانی از تمام مرورگرهای مدرن

## نحوه استفاده

1. فایل `index.html` را باز کنید
2. تمام استایل‌ها و اسکریپت‌ها از طریق لینک‌های موجود در فایل بارگذاری می‌شوند
3. برای شخصی‌سازی، فایل‌های مربوطه در پوشه `src` را ویرایش کنید

## شخصی‌سازی

برای شخصی‌سازی پورتفولیو:

- در فایل `index.html` نام، اطلاعات تماس و محتوای متنی را تغییر دهید
- در فایل `src/js/main.js` متن تایپ‌رایتر و عملکردهای سفارشی را ویرایش کنید
- در فایل `src/styles/main.css` رنگ‌ها و استایل‌های سفارشی را اضافه کنید

## مزایای این پروژه

- **کدنویسی تمیز**: ساختار منظم و کامنت‌های فارسی
- **قابلیت نگهداری**: جدا کردن کدهای HTML، CSS و JS برای راحتی در نگهداری
- **بهینه‌سازی**: کاهش حجم کد و بهبود عملکرد
- **طراحی مدرن**: رابط کاربری زیبا و حرفه‌ای
- **تجربه کاربری عالی**: تعاملات هوشمند و نرم

این پروژه می‌تواند به عنوان نمونه کار قدرتمند برای استخدام یا نمایش مهارت‌های فرانت‌اند استفاده شود.
