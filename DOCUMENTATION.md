# مستندات پروژه پورتفولیو

## معرفی

این پروژه یک پورتفولیو حرفه‌ای و مدرن برای نمایش مهارت‌ها و پروژه‌های یک توسعه‌دهنده فرانت‌اند است. طراحی شده با تمرکز بر تجربه کاربری عالی، طراحی واکنش‌گرا و ویژگی‌های پیشرفته.

## ویژگی‌های کلیدی

### 1. طراحی واکنش‌گرا
- سازگار با تمام اندازه‌های صفحه نمایش
- استفاده از Tailwind CSS برای مدیریت استایل‌ها
- توجه ویژه به UX/UI

### 2. تم تاریک/روشن
- قابلیت تغییر تم با ذخیره در localStorage
- تجربه کاربری سفارشی بر اساس ترجیحات کاربر

### 3. انیمیشن‌های تعاملی
- استفاده از CSS Animations برای ایجاد حرکت‌های نرم
- افکت‌های تعاملی برای بهبود تجربه کاربری

### 4. کرسر سفارشی
- طراحی کرسر تعاملی با افکت مغناطیسی
- بهبود تعامل کاربر با رابط

### 5. سیستم مهارت‌ها
- نمایش گرافیکی مهارت‌ها با انیمیشن
- نوارهای پیشرفت تعاملی

## ساختار پروژه

```
/workspace/
├── index.html              # فایل اصلی HTML
├── src/                    # پوشه منابع پروژه
│   ├── js/                 # اسکریپت‌های جاوااسکریپت
│   │   └── main.js         # تمام کدهای جاوااسکریپت
│   ├── styles/             # فایل‌های استایل
│   │   ├── input.css       # ورودی Tailwind
│   │   └── main.css        # خروجی Tailwind (تولید شده)
│   └── assets/             # فایل‌های دارایی (تصاویر و ...)
├── node_modules/           # ماژول‌های npm
├── package.json            # وابستگی‌ها و اسکریپت‌ها
├── tailwind.config.js      # پیکربندی Tailwind
├── README.md               # توضیحات کلی پروژه
├── DOCUMENTATION.md        # مستندات جامع
└── .gitignore              # فایل‌های نادیده گرفته شده
```

## نصب و راه‌اندازی

### پیش‌نیازها
- Node.js (نسخه 14 یا بالاتر)
- npm یا yarn

### مراحل نصب

1. کلون کردن یا دانلود پروژه
2. نصب وابستگی‌ها:
   ```bash
   npm install
   ```
3. تولید فایل استایل:
   ```bash
   npm run build
   ```
4. یا برای توسعه با نظارت بر تغییرات:
   ```bash
   npm run dev
   ```
5. اجرا:
   ```bash
   npm start
   ```

## شخصی‌سازی

### تغییر اطلاعات شخصی
1. ویرایش `index.html` برای تغییر نام، شغل و اطلاعات تماس
2. ویرایش `src/js/main.js` برای تغییر متن تایپ‌رایتر
3. ویرایش `README.md` برای توضیحات جدید

### تغییر رنگ‌ها
1. ویرایش `tailwind.config.js` برای تغییر متغیرهای رنگ
2. اجرای دوباره دستورات تولید استایل

### اضافه کردن پروژه‌ها
1. ویرایش بخش پروژه‌ها در `index.html`
2. اضافه کردن اطلاعات جدید در ساختار موجود

## توسعه

### استفاده از Tailwind CSS
- تمام کلاس‌های Tailwind در فایل `input.css` وارد شده‌اند
- پیکربندی در `tailwind.config.js` قابل ویرایش است
- خروجی نهایی در `main.css` تولید می‌شود

### مدیریت اسکریپت‌ها
- تمام کدهای JavaScript در `main.js` سازمان‌دهی شده‌اند
- استفاده از کلاس‌ها برای ساختاردهی بهتر
- کامنت‌های فارسی برای راحتی درک

## بهینه‌سازی‌ها
